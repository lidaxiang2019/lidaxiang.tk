1. 反射机制允许程序在运行时取得任何一个已知名称的class的内部信息，容许程序在运行时加载、探知、使用编译期间未知的class。

即Java的反射机制可以加载一个运行时才得知名称的class，获得其完整结构。比如父类中方法需要调用子类的泛型作为参数的时候。
